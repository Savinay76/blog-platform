# Web Performance Tips 2024

Performance is key to user experience and SEO.

## Top Tips

1.  **Optimize Images**: Use WebP/AVIF formats.
2.  **Code Splitting**: Load only what you need.
3.  **Caching**: Use CDNs effectively.
4.  **Reduce JavaScript**: Keep your bundle size low.

> "Performance is a feature."

Make sure to measure with Lighthouse!
